# AutoPart Pro - API Backend 🚀

Sistema de gestión y optimización de inventario para tiendas de repuestos automotrices. Desarrollado con un enfoque de arquitectura escalable, segura y prácticas DevOps.

## 🛠️ Tecnologías Utilizadas

* **Runtime:** Node.js (v18+)
* **Framework:** Express.js
* **Base de Datos:** MySQL (Relacional)
* **Control de Versiones:** Git & GitHub

---

## 🏗️ Arquitectura del Proyecto

El backend está construido bajo el patrón de diseño **MVC (Modelo-Vista-Controlador)**, separando las responsabilidades de acceso a datos, lógica de negocio y respuestas API (JSON), montado sobre una infraestructura **Cliente-Servidor**.
